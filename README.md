# Halka# Halka GitHub Pages Website

A bilingual Bosnian-English static website for Halka.

## Publish on GitHub Pages

1. Create a new public GitHub repository, for example `halka`.
2. Upload `index.html`, the `assets` folder, and this README file.
3. Open the repository's **Settings**.
4. Choose **Pages** in the left menu.
5. Under **Build and deployment**, select **Deploy from a branch**.
6. Choose the `main` branch and `/root`, then click **Save**.
7. GitHub will provide a live address such as `https://yourusername.github.io/halka/`.

## Before publishing

Replace these placeholders in `index.html`:

- `halka@rabata.org`
- `@halka.ba`
- Empty social media and library links marked with `href="#"`

You can also replace the circular H mark with the official Halka logo later.
